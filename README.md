# GWS-Auto-for-Android

ローコードで実現する、あなたのビジネス自動化パートナー

## 概要

`GWS-Auto-for-Android`は、Google Workspace（GWS）との連携と定型業務の自動化を、ローコードで実現するAndroidアプリケーションです。

従来、Google Apps Script (GAS) などプログラミングの知識が必要だった業務自動化を、スマートフォンに最適化された直感的なUIで提供します。個人事業主や従業員の少ない零細企業など、事務作業に多くのコストをかけられないビジネス層をターゲットに、誰でも簡単に導入できるRPA（Robotic Process Automation）環境を提供し、ビジネスの効率化と生産性の向上を支援します。

## 主な機能

### 1. モジュールによるワークフロー構築
GASではコード記述が必要だった処理を、一連の動作をまとめた「モジュール」として提供します。ユーザーはこれらのモジュールを自由に組み合わせることで、複雑な一連の動作を「ワークフロー」として簡単に構築できます。

*   **モジュール例:**
    *   スプレッドシートからPDFファイルを生成する
    *   Excelデータをスプレッドシートに変換する
    *   Googleフォームの回答をトリガーに、カレンダーに予定を登録する

### 2. 高度なワークフロー設計
条件分岐や処理のグループ化など、複雑なロジックをGUIで直感的に組み立てることができ、単純な繰り返し作業だけでなく、状況に応じた高度な自動化を実現します。

### 3. 柔軟な自動実行トリガー
構築したワークフローは、様々なトリガーによって自動実行が可能です。

*   **予約実行:** 指定した日時に一度だけ実行
*   **繰り返し実行:** 毎日、毎週、毎月など、定期的なスケジュールで実行

### 4. ホーム画面ショートカット
手動で実行したいワークフローは、スマートフォンのホーム画面にショートカットとして配置できます。これにより、必要な時にワンタップで複雑な業務を即座に実行できます。

### 5. 視覚的なスケジュール管理
登録したトリガーのスケジュールは、シンプルな一覧形式だけでなく、カレンダー形式など複数のビューで視覚的に確認でき、実行計画の管理が容易です。

### 6. セキュアな認証とデータ管理
*   **Googleアカウント認証:** Googleアカウントで安全にログインし、ユーザーごとの設定やワークフローをクラウドに保管します。
*   **シークレットキー管理:** APIキーなどの機密情報を安全に保存し、ワークフロー内では「キーの名称」を指定するだけで安全に利用できます。

### 7. 汎用的なモジュール設定
各モジュールは、URL、シート名、セルの範囲など、動作に必要な値をユーザーが任意に設定できます。これにより、汎用的なモジュールを最小限の設定で様々な業務に活用できます。

## 目的と提供価値

*   **業務効率化:** スケジュールの決まったルーティン業務や、手作業で実行するとミスが発生しやすい業務を自動化し、作業時間の短縮と精度の担保を実現します。
*   **属人化の抑止:** GUIベースで構築されたワークフローは、業務の流れを可視化し、担当者以外でも内容の理解やメンテナンスが容易になります。これにより、業務の属人化を防ぎ、組織全体の生産性を向上させます。
*   **RPAの民主化:** 高価で専門知識が必要だったRPAを、Androidスマートフォンという最も身近なデバイスで、誰でも手軽に利用できる環境を提供します。

## 今後の展望

将来的には、PCのブラウザから同じデータにアクセスし、ワークフローの構築や管理ができるWebアプリケーションとの連携を視野に入れています。
